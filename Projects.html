Mini Project

Biometric Fingerprint Voting Machine Using Arduino

In this project we have built a Working model of Biometric Fingerprint Based Voting Machine which uses RF-ID card for Initialization of the machine and then Voter Can vote using his finger print the machine will allow the voter if it finds the match to the fingerprint in the database






Major Project

Fire Fighting Robot With Two Modes Of Operation Using Arduino

In This Project we have built a model of Fire Fighting Robot which works in Two Modes of operation Automatic and manual so that it can be used according to the situations and it is a very efficient model for fire detection and extinguishing. In automatic mode it continuosly look for fire and extinguish the fire automatically if detected and in manual mode we can control the Robot using mobile app to handle the situation Manually






Internship Project

Energy Efficient Smart Street Light For Smart City Using STM32F429ZIT6 Microcontroller

Smart street lights is a project on intelligent illumination control of street lights to optimize the problem of power consumption and illumination of the streets, late in the night. we have designed a movement detection based street light system. The lights turn on before pedestrians and vehicles come and turn off or reduce power when there is no one. Hence Reducing the power consumption.
